📊 Concepts of Probability

🎲 Experiment & Event

Experiment: process with well‑defined outcomes.
Sample space \(S\): set of all possible outcomes.
Event \(E\): subset of \(S\).

\( \text{Pr}(E) = \frac{\# \text{outcomes in } E}{\# \text{outcomes in } S} \) (equally likely)

📘 eg. Toss 2 coins: \(S=\{HH,HT,TH,TT\}\), \(E=\{\text{at least 1 H}\} = \{HH,HT,TH\}\)

✔ understood
📈 Prob. definitions

Relative frequency (empirical):

\( \displaystyle \text{Pr}(A) \approx \frac{\text{count of } A}{\text{total trials}} \)

Classical (equally likely):

\( \displaystyle \text{Pr}(A) = \frac{\# \text{outcomes in } A}{\# \text{outcomes in } S} \)

🧮 eg. ACT guess (5 choices): \(\text{Pr(wrong)} = \frac{4}{5}=0.8\)

✔ mastered
📋 Contingency tables

Classify outcomes in rows/columns. Joint & marginal probabilities.

\( \text{Pr}(A \cap B) = \frac{\text{cell count}}{\text{total}} \)  ·  \( \text{Pr}(A) = \frac{\text{row/col total}}{\text{total}} \)

📊 eg. 223 sales: 96 Windows → \(\text{Pr(Windows)} = 96/223 = 0.4305\)

✔ can do
🔢 Discrete R.V.

Countable values (finite or infinite sequence). Described by probability distribution \(p(x)\).

\( \sum p(x) = 1 \),   \( 0 \le p(x) \le 1 \)

🎯 eg. \(X\) = # heads in 2 tosses: \(P(0)=1/4, P(1)=1/2, P(2)=1/4\)

✔ got it
📉 Continuous R.V.

Uncountably infinite values. Described by probability density function \(f(x)\).

\( f(x) \ge 0 \),   The area between the curve and x-axis = 1.

📏 eg. height, weight, time. \(\text{Pr}(X=a)=0\) for any exact \(a\).

✔ clear
📐 Uniform distribution

Constant density on \([a,b]\): \(f(x) = \frac{1}{b-a}\) for \(a \le x \le b\).

\( f(x) = \begin{cases} \frac{1}{b-a}, & a \le x \le b \\ 0, & \text{otherwise} \end{cases} \)

⏳ eg. wait time uniform on [0,5] min: \(\text{Pr}(<3) = \frac{3-0}{5-0} = 0.6\)

✔ familiar
⚡ Events & probabilities

For discrete R.V., events are subsets of possible values.
\(E_1 = \{X=1\}\), \(E_2 = \{X>0\}\), etc.

\( \text{Pr}(X \in A) = \sum_{x \in A} p(x) \)

🧩 eg. 3 children, \(X\)=#boys. \(\text{Pr}(X=2) = 3/8\)

✔ understood
📊 Continuous events

Events are intervals. Probability = area under density curve.

\( \text{Pr}(a \le X \le b) = \int_{a}^{b} f(x)\,dx \)

🎯 eg. uniform [1,9]: \(\text{Pr}(X>5) = \frac{9-5}{9-1} = 0.5\)

✔ got it
🚆 Shuttle train

Arrival time uniform [0,5] min.

\( \text{Pr}(\text{wait}<3) = \frac{3}{5} = 0.6 \)
\( \text{Pr}(\text{wait}=3) = 0 \)
\( \text{Pr}(\text{wait}>3) = \frac{2}{5} = 0.4 \)

⏱️ exact time has zero probability for continuous R.V.

✔ solved
✅ Distribution checklist

Discrete:

\(\sum p(x)=1\), \(0 \le p(x) \le 1\)

Continuous:

\(f(x) \ge 0\), The area under the curve = 1.

🔍 eg. \(f(x)=1/8\) on [1,9] is valid (area = 1).

✔ confirmed
