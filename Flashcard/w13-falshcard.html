📊 Correlation & Least‑Squares Regression

📈 Correlation coefficient r

Concept: measures strength and direction of linear relationship between two numeric variables.

$r = \dfrac{SS_{xy}}{\sqrt{SS_{xx}}\sqrt{SS_{yy}}}$

Interpretation: $r$ ∈ [-1,1] ; $|r|$ close to 1 → strong linear association.

Example: Height vs. Weight: $r = 0.789$ → moderate positive correlation.

key
📉 Regression line $\hat{y} = b + mx$

Concept: describes how response $y$ changes with predictor $x$. Slope $m$, intercept $b$.

$m = \dfrac{SS_{xy}}{SS_{xx}} \quad , \quad b = \bar{y} - m\bar{x}$

Example: Beer: $\hat{y} = 25 + 26.3x$ (calories vs. alcohol %). Slope: +26.3 cal per 1% alcohol.

formula
📊 $R^2$ & goodness-of-fit explained variation

Concept: proportion of total variation in $y$ explained by the regression model.

$R^2 = \dfrac{SSR}{SST} = r^2$

Interpretation: e.g., $R^2 = 0.64$ → 64% of variation in $y$ is explained by $x$.

Example: if $r = 0.8$, then $R^2 = 0.64$ → good linear fit.

goodness
🔍 Inference for slope $m$

Concept: test if slope $m \neq 0$ → significant linear correlation.

$H_0: m = 0 \quad \text{vs.} \quad H_a: m \neq 0$

Interpretation: if 0 not in CI for $m$, variables are significantly correlated.

Example: 95% CI for slope: [1.2 , 4.5] → 0 not included → significant positive correlation.

inference
📐 Sum of squares building blocks

Concept: used for correlation and regression coefficients.

$SS_{xx} = \sum (x_i-\bar{x})^2$
$SS_{yy} = \sum (y_i-\bar{y})^2$
$SS_{xy} = \sum (x_i-\bar{x})(y_i-\bar{y})$

Example: from height–weight data: $SS_{xy}$ used to compute $r$ and $m$.

calc
🔮 Prediction with regression $\hat{y}$

Concept: use fitted line to predict $y$ for a given $x$.

$\hat{y} = b + m x$

Example: stream flow: $\hat{y} = 1.6 + 29x$. For $x = 0.45$ inches → $\hat{y} = 14.65$ gal/min.

Interpretation: prediction is reliable only within range of data.
