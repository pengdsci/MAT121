📊 Frequency Tables & Graphs Review Flash Cards

📌 Statistics def

Statistics is the science of collecting, organizing, visualizing, analyzing, and interpreting data to make decisions.

📊 Data → Information → Decision
✔️ I know this
👥 Population / Sample

Population: all outcomes, responses, or counts of interest.
Sample: a subset of the population.

🧮 Parameter = population characteristic
Statistic = sample characteristic
✔️ I know this
🏷️ Data Types

Qualitative: Nominal (unordered) · Ordinal (ranked)
Quantitative: Discrete (counts) · Continuous (measurements)

📌 Examples: eye color (nominal), grade A–F (ordinal), shoe size (discrete), height (continuous)
✔️ I know this
📋 Freq. Table (cat.)

Lists categories (left) and frequencies (right).

$ \text{relative frequency} = \frac{\text{frequency}}{\text{total}} $
$ \text{cumulative freq.} = \text{sum of frequencies up to that category}$
✔️ I know this
📊 Bar Chart

Geometric representation of categorical frequencies. Bars can be rearranged in ascending/descending order to create a Pareto chart.

📈 height = frequency (or relative frequency)
✔️ I know this
🥧 Pie Chart

Each slice’s central angle: $ \text{angle} = \text{relative freq.} \times 360^\circ $

📐 $ \text{degrees} = \frac{\text{frequency}}{\text{total}} \times 360 $
✔️ I know this
📉 Histogram

For numerical data: bins (equal width) on x‑axis, frequencies on y‑axis. No gaps between bars.

📦 Bin width = (extended max – min) / #bins
boundaries: [low, high) or (low, high]
✔️ I know this
🔢 Bin Construction

1. Choose #bins
2. Extend data window to convenient endpoints
3. Equal width = (extended width)/#bins
4. Count frequencies per bin

📌 Example: [20,31), [31,42), …
✔️ I know this
🔍 Descriptive vs Inferential

Descriptive: organize, summarize, display (tables, charts, averages).
Inferential: use sample to make conclusions about population.

🧪 Descriptive = summarize data · Inferential = generalize
✔️ I know this
🎯 Parameter vs Statistic

Parameter: numeric characteristic of a population.
Statistic: numeric characteristic of a sample.

📊 e.g., population mean μ vs sample mean x̄
✔️ I know this
📈 Cumulative Freq.

For ordered categories, cumulative frequency adds frequencies of all categories up to that point.

📉 Example: F, D or below, C or below, …
✔️ I know this
📊 Relative & Cumulative

Relative frequency = frequency / total.
Cumulative relative frequency = cumulative frequency / total.

$ \text{Rel.} = \frac{f}{n} $  ·  $ \text{Cum. Rel.} = \frac{\text{cum.}f}{n} $
✔️ I know this
