📈 Numerical Measures

Mean \(\bar{x}, \mu\)

Average of all values. Sensitive to outliers.

\(\bar{x} = \frac{\sum x_i}{n}\)   (sample)
\(\mu = \frac{\sum x_i}{N}\)   (population)

Example: \(\{1,4,7\}\) → mean = \( (1+4+7)/3 = 4 \)

center
Median \(Q_2\)

Middle value in sorted data. If even, average of two middle.

odd \(n\): middle position \((n+1)/2\)
even \(n\): avg of \(n/2\) and \(n/2+1\)

Example: \(\{1,2,6,7\}\) → median \(= (2+6)/2 = 4\)

center
Mode highest frequency

Most frequent value. Unimodal / bimodal / multimodal.

no formula — count frequencies

Example: \(\{7,7,9,9,9,10\}\) → mode = 9

center
Variance \(s^2, \sigma^2\)

Average squared deviation from the mean.

\(s^2 = \frac{\sum (x_i - \bar{x})^2}{n-1}\)   (sample)
\(\sigma^2 = \frac{\sum (x_i - \mu)^2}{N}\)   (population)

Example (sample): \(\{1,4,7\}\) → \(s^2 = 9\)

variation
Std. Deviation \(s, \sigma\)

Square root of variance. Same units as data.

\(s = \sqrt{s^2}\)   \(\sigma = \sqrt{\sigma^2}\)

Example: \(\{1,4,7\}\) → \(s = \sqrt{9} = 3\)

variation
z-score standardized

Number of standard deviations from the mean.

\(z = \frac{x - \bar{x}}{s}\)   (sample)
\(z = \frac{x - \mu}{\sigma}\)   (population)

Example: \(\{1,4,7\}\) → \(z\)-scores: \(-1, 0, 1\)

location
Percentile \(P_k\)

% of values below a given score.

\(L = \frac{k}{100} \cdot n\)
if \(L\) integer: avg of \(L\) and \(L+1\)
if not: round up

Example: sorted 35 values → \(P_{55}\) = 10

location
5‑Number Summary

Min, Q1, Q2, Q3, Max (box‑plot).

\(Q_1 = P_{25}, \quad Q_2 = P_{50}, \quad Q_3 = P_{75}\)

Example (CD data): 20.5, 39, 51, 56.5, 74.5

summary
IQR interquartile range

Middle 50% spread – robust to outliers.

\(\text{IQR} = Q_3 - Q_1\)

Example: \(56.5 - 39 = 17.5\) (CD data)

variation
Box‑plot visual 5‑number

Graphical summary of distribution shape.

box from \(Q_1\) to \(Q_3\), whiskers to min/max

Interpret: skewness, outliers, spread.

summary
Σ Notation summation

Sum of all values in a data set.

\(\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n\)

Example: \(\{1,4,7\}\) → \(\sum x_i = 12\)

notation
Parameter vs Statistic

Greek for population, English for sample.

\(\mu\) (pop mean)   vs   \(\bar{x}\) (sample mean)
\(\sigma^2\) (pop var)   vs   \(s^2\) (sample var)

Example: sample mean \(\bar{x}=4\) vs population mean \(\mu\)
